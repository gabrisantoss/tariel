# Banco de Referencias Externas · NR12

Base local de apoio para a familia `NR12`, separada do material real de operacao da empresa.

## Objetivo

- consolidar fontes oficiais e publicas para estudo da norma;
- acumular imagens de contexto tecnico para ajudar no entendimento de maquina, protecao, intertravamento e zona de risco;
- preparar um banco inicial para orientar `schema -> report pack -> projection -> PDF`.

## Regra de uso

- esta pasta nao substitui laudos reais da operacao;
- fotos publicas servem como referencia visual, nao como prova normativa nem como exemplo oficial de layout de laudo;
- a fonte primaria para regra tecnica continua sendo o material oficial do `MTE` e os exemplos reais da empresa;
- exemplos de internet com licenca aberta podem ajudar no contexto visual, mas nao devem virar contrato de preenchimento por si so.
