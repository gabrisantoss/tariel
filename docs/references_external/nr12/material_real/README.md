# Material Real · NR12

Esta pasta fica reservada para os arquivos reais da operacao da empresa relacionados a `NR12`.

## Regra de separacao

- `references_external/` = base oficial e publica
- `material_real/` = arquivos reais da empresa

Essa separacao existe para evitar misturar:

- referencia normativa;
- referencia visual publica;
- evidencias operacionais reais.

## Quando os arquivos chegarem

Organizar por subpastas:

- `laudos_excel/`
- `laudos_pdf/`
- `fotos/`
- `audios_transcritos/`
- `observacoes_operacionais/`

## Metadados minimos recomendados por caso

- codigo do caso
- unidade
- tipo de maquina/equipamento
- tipo de entrega
- status final
- data da vistoria
- observacao principal

## Objetivo

Usar esse material para descobrir:

- qual subfamilia da `NR12` e mais recorrente;
- qual estrutura de laudo realmente se repete;
- qual recorte vale virar contrato oficial no produto.
